# Changelog

All notable changes to this project will be documented in this file.

## V2.0.1

### Added

- Added `人生回溯 / Life Review` as a new guided flow.
- Added `未来指引 / Future Guide` as a new guided flow.
- Added scenario-based comparison logic between current, past, and future results.
- Added version/testing notes describing the current empirical-validation status.

### Changed

- Reworked the result page to better support bilingual display and scenario comparison.
- Improved the history page so saved results can be opened and reviewed more directly.
- Refined homepage entry points for the new time-perspective features.
- Improved global layout messaging so version notes are easier to see without crowding navigation.

### Fixed

- Fixed several mixed Chinese/English display issues.
- Fixed duplicated or awkward nearby labels in the time-comparison UI.
- Fixed parts of the result-viewing flow that were inconsistent for historical and scenario-based results.

### Notes

- `Psyching V2.0.1` is currently a testing version.
- Reliability, validity, and item-parameter tuning are planned for a later empirical stage.
